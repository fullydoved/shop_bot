Reader Setup

Prerequisites

ACR1252U USB NFC reader, Python 3

Install

pip install pyscard ndeflib requests

Scan mode

Polls for tags and opens the bin page in your browser:

python reader.py

Write mode

Prompts for bin codes in a loop, writing an NDEF URL to each tag:

python reader.py --write

Configuration

Set NFC_HOST env var to point at your server (defaults to http://localhost:8000):

export NFC_HOST=http://192.168.1.50:8000
